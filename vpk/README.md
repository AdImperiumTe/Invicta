# Python3 VPK Scripts
Allows the user to extract or make .vpk files with Python.

### Install the required modules
```shell
cd path/to/this/directory
pip install -r requirements.txt
```

## Extract VPK
### Windows
```shell
python ExtractVPK.py
```
### Linux
```bash
python3 ExtractVPK.py
```

## Make VPK
### Windows
```shell
python MakeVPK.py
```
### Linux
```bash
python3 MakeVPK.py
```
